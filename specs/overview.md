# アプリ概要

## 目的

「マイクラ人狼」専用の Bedrock Dedicated Server (BDS) を構築・起動するための、Windows 専用ランチャーアプリ。

## スコープ

- **対象OS**: Windows専用。Linuxでサーバーを建てたいユーザー向けには別アプリを用意する(本アプリのスコープ外)。
- **副次機能**: 人狼専用ワールドの構築が主目的だが、副作用として簡易的な任意ワールドを建てる仕組みも付与するかもしれない(詳細未定)。

## 前提となる外部システム

- **公式Linuxサーバー**(未構築)
  - アドオン等のバージョン管理・配信
  - プレイヤーのRankなどのアカウント管理
  - 本アプリは起動時にこのサーバーへ問い合わせを行う

## 主な機能

### 1. BDSサーバーの構築・起動

マイクラ人狼に必要な複数アドオンを導入した状態のワールドを構築する。必要アドオン:

- game-manager
- vanillapack
- additional-roles-1
- kairo
- kairo-database

### 2. 自動アップデート

アプリ起動時に公式サーバーへ更新情報を問い合わせ、更新があれば上記アドオン等を自動的に最新化する。

### 3. 専用BDSアドオン

本アプリ専用のBDSアドオンを新規に作成する。

- アプリには内包(バンドル)する
- リポジトリはこの `bds-launcher` とは別に管理する(TODO: リポジトリ名・役割の詳細)

### 4. (副次) 簡易ワールド構築

人狼以外の任意ワールドも簡易に構築できる仕組みを検討中(詳細未定)。

## 非スコープ

- Linux版サーバー構築は別アプリで対応する(本アプリでは扱わない)
